# Build your first Embeddable

Create your first Embeddable dashboard.

<Callout emoji="üí°">
  An **Embeddable** is an interactive dashboard that you embed in your web
  application.
</Callout>

<Steps>

### Create a New Embeddable

On your workspace homepage, click **Create new Embeddable** to create a fresh dashboard.

### Add a Dataset

Datasets are how you access data on a dashboard.

Your workspace is linked to our sample database.

<VideoComponent
  src="/video/create-new-dataset-example.mp4"
  width="1100"
  height="746"
/>

1. In the **Datasets Panel** in the bottom right, click the **+** icon.
2. Name your dataset.
3. Select **My customers** (or any other sample data model), then **Save**.

### Add a Donut Chart

Build your dashboard by adding components to the canvas.

<VideoComponent src="/video/create-pie-chart.mp4" width="1100" height="746" />

1. Click **Add component**. Pick **Donut Chart**.
2. In the edit panel, set **Dataset to display** to the dataset you just created.
3. Select your preferred **Measure** and **Dimension**.
4. Hit **Save**.

<Callout emoji="‚ö°Ô∏è">
  **Bonus**: check the **console** in the right sidebar to see what's happening
  behind the scenes.
</Callout>

### Add Interactivity

Adding interactivity is quick and easy.

<VideoComponent
  src="/video/Add-interactivity-remarkable.mp4"
  width="1100"
  height="746"
/>

1. Click **Add component**, choose **Date Range Select Field**, then hit **Save**.

<Callout emoji="‚ö°Ô∏è">
  In the right-hand Variables panel in the Builder, you‚Äôll see a new variable
  appear: **date-range value**. Variables store outputs from interactive
  components (like date pickers) and can be passed around the dashboard.
</Callout>

2. Edit your **Donut Chart**.
3. Click **Filter** (under the Dataset input) and, from the dropdown, choose the **Signed up at** Dimension.
4. Next to the filter operator (‚ÄúIn date range‚Äù), click the **link** icon.
5. Select the **date-range value** variable, then hit **Save**.

</Steps>

<Callout emoji="ü•≥">
  **Success!** Switch to Preview mode in the top bar to test the interaction.
</Callout>
